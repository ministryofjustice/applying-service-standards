# Apply the Service Standard

Guidance and information for working to the Government Service Standard in the Ministry of Justice.


## Prerequisites

Docker

## Then run the app

Run `make run`

## Access points

Hot reload version (BrowserSync)<br>
**http://localhost:3000**

The application<br>
**http://localhost:3052**

BrowserSync UI<br>
**http://localhost:3001**
